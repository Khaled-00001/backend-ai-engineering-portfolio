# AI Workflow Audit and Tool Setup

## Overview

This workflow audit analyzes my recurring weekly tasks and identifies where AI can help, where human judgment is essential, and where automation can improve productivity without reducing learning quality.

---

# Workflow Audit

| Task | Classification | Why |
|------|----------------|-----|
| Solving competitive programming problems | Collaborate with AI | AI helps explain algorithms and optimize approaches, but I write and debug the final solution myself. |
| Learning Spring Boot | Delegate to AI with review | AI summarizes documentation and generates examples that I verify by building projects. |
| Building backend APIs | Collaborate with AI | AI accelerates development, while I make architecture and implementation decisions. |
| Debugging Java applications | Collaborate with AI | AI suggests possible causes, but I validate every fix manually. |
| Writing SQL queries | Delegate to AI with review | AI can generate queries faster, but I always test and optimize them. |
| Planning my weekly study schedule | Fully automate | AI can generate an efficient study plan based on my goals and deadlines. |
| Writing GitHub README files | Delegate to AI with review | AI creates a strong first draft, and I edit it for accuracy and clarity. |
| Learning system design concepts | Collaborate with AI | AI acts as a tutor while I study and apply the concepts myself. |
| Preparing internship applications | Collaborate with AI | AI improves wording, but I personalize every application before submitting it. |
| Managing Git and GitHub workflow | Delegate to AI with review | AI reminds me of Git workflows and best practices, but I perform every Git operation myself. |
| Technical interviews | Just Me | Communication, confidence, and technical reasoning must be entirely my own. |
| Coding assessments | Just Me | They measure my real programming skills and problem-solving ability. |
